
# Policonnect

###  Descripción del Proyecto

**Policonnect** es una plataforma web diseñada para conectar a estudiantes, docentes y autoridades de universidades del Ecuador, facilitando el acceso y la interacción con contenido académico en un solo lugar.

El sistema integra dos módulos principales:

-  Repositorio Académico: Permite subir, organizar y consultar documentos como tesis, investigaciones y archivos técnicos.
-  Sistema de Feedback Social: Un espacio tipo red social donde los usuarios pueden compartir ideas, comentarios y sugerencias.





##  Funcionalidades Principales

###  Repositorio de Documentos
- Subida de archivos (PDF, DOCX, etc.)
- Organización por categorías (tesis, informes, investigaciones)
- Búsqueda por título, autor o universidad
- Descarga de documentos

###  Sistema de Feedback
- Publicación de posts (ideas, comentarios, sugerencias)
- Sistema de likes y comentarios
- Feed dinámico
- Interacción entre usuarios

###  Gestión de Usuarios
- Registro e inicio de sesión
- Perfiles de usuario
- Edición de información personal

###  Notificaciones
- Alertas por interacciones (likes, comentarios)
- Actividad relevante dentro de la plataforma
--##  Stack Tecnológico

### Frontend
- React

- CSS / Tailwind (opcional)

### Backend
- Node.js
- Express.js

### Base de Datos
- PostgreSQL (principal)
- Firebase o Supabase (alternativas)
